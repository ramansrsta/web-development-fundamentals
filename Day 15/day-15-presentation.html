Javascript Introduction

JavaScript is a high-level, interpreted programming language that conforms to the ECMAScript specification. It is a language that is also characterized as dynamic, weakly typed, prototype-based, and multi-paradigm.

  • JavaScript is a dynamic, high-level, interpreted language.
  • It enables web pages to be interactive and responsive.
  • Used in Client Side, Server Side, Mobile and Desktop Applications

Vairables

  • var: Function-scoped, older standard.
  • let: Block-scoped, modern standard.
  • const: Block-scoped, cannot be reassigned.
var globalVar = "I am var"; let blockVar = "I am let"; const constVar = "I am constant"; console.log(globalVar, blockVar, constVar);

Variables Scope

  • Global Scope: Accessible everywhere.
  • Function Scope: Inside functions only.
  • Block Scope: Defined by {} with let and const.
let globalVar = "Global"; function scopeExample() { let localVar = "Local"; console.log(globalVar); // Accessible console.log(localVar); // Accessible } scopeExample(); console.log(localVar); // Error

Booleans and Comparison

  • Boolean: true or false.
  • Comparison Operators:
    • == (loose equality)
    • === (strict equality)
    • != (not equal)
    • !== (strict not equal)
    • > (greater than)
    • < (less than)
    • >= (greater than or equal)
    • <= (less than or equal)
let isTrue = true; let isEqual = 5 == "5"; console.log(isEqual); // true let isStrictEqual = 5 === "5"; console.log(isStrictEqual); // false

Null and Undefined

  • null: Represents an intentional absence of value.
  • undefined: Represents an uninitialized variable.
let empty = null; let undef; console.log(empty); // null console.log(undef); // undefined

Objects and Arrays

  • Objects: Key-value pairs.
  • Arrays: Ordered lists of values.
let person = { name: "John", age: 30 }; let fruits = ["apple", "banana", "orange"]; console.log(person.name); // "John" console.log(fruits[1]); // "banana" // Adding a new key-value pair person.job = "Developer"; console.log(person.job); // "Developer" // Adding a new item to the array fruits.push("grapes"); console.log(fruits); // ["apple", "banana", "orange", "grapes"]
