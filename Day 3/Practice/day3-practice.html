1. HTML Forms

✅ Best Practices

  • Always use <label> elements with for attributes
  • Group related form elements using <fieldset> and <legend>
  • Include proper form validation attributes
  • Use semantic input types
  • Provide clear feedback for form validation

❌ What to Avoid

  • Don't use placeholder text as labels
  • Don't skip form validation
  • Don't use generic input types when specific ones exist
  • Don't forget to style focus states
  • Don't create overly long forms without sections

2. HTML Tables

✅ Best Practices

  • Use proper table structure (<thead>, <tbody>, <tfoot>)
  • Include table headers with <th> elements
  • Use scope attribute for header cells
  • Add captions with <caption> element

❌ What to Avoid

  • Don't use tables for layout purposes
  • Don't nest tables unnecessarily
  • Don't skip table headers
  • Don't create overly complex tables
