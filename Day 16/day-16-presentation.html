ES6+ Introduction

ES6 (ECMAScript 2015) introduced modern features for cleaner, faster, and more efficient JavaScript development.

  • Template Literals
  • Arrow Functions
  • Destructuring
{`// Template Literals Example let name = "John"; console.log(\`Hello, \${name}!\`); // Output: Hello, John `}

Destructuring

Destructuring allows unpacking values from arrays or properties from objects into distinct variables.

{`// Destructuring Arrays const arr = [1, 2, 3]; const [first, second] = arr; console.log(first, second); // Output: 1 2 // Destructuring Objects const person = { name: "Alice", age: 30 }; const { name, age } = person; console.log(name, age); // Output: Alice 30 `}

Spread and Rest Operators

The spread operator (...) expands elements, while the rest operator collects arguments into an array.

{`// Spread Example const arr1 = [1, 2]; const arr2 = [...arr1, 3, 4]; console.log(arr2); // Output: [1, 2, 3, 4] // Rest Example function sum(...nums) { return nums.reduce((total, n) => total + n, 0); } console.log(sum(1, 2, 3)); // Output: 6 `}

Promises

Promises help manage asynchronous operations effectively.

{`// Promise Example const fetchData = () => new Promise((resolve, reject) => { setTimeout(() => resolve("Data fetched"), 1000); }); fetchData().then(data => console.log(data)); // Output after 1s: Data fetched `}

Async/Await

Async/Await simplifies working with Promises for asynchronous code.

{`// Async/Await Example const fetchData = async () => { const data = await new Promise(resolve => setTimeout(() => resolve("Data fetched"), 1000)); console.log(data); }; fetchData(); // Output after 1s: Data fetched `}
